# Neuroscore

Un'applicazione dimostrativa per lo scoring semi-automatico dei test neuropsicologici in italiano.

L'applicazione lavora interamente lato client e salva i dati dei pazienti e i punteggi dei test nella memoria locale del browser. Nessun dato viene trasmesso in rete.

La versione più recente è disponibile [qui](https://sapienza-neuropsicologia.github.io/neuroscore/).
